布局管理器
  • 布局管理器可以决定组件在容器中的布局方式，避免了手动设置每个组件的位置和大小，从而简化了GUI设计过程

常见的布局管理器

什么是布局管理器？

布局管理器（Layout Manager）是Java Swing中用于控制容器中组件排列和大小的对象。它们决定了组件在容器中的布局方式，避免了手动设置每个组件的位置和大小，从而简化了GUI设计过程。

FlowLayout

  • 简介: FlowLayout 是最简单的布局管理器，它按水平方向从左到右排列组件，当一行排满时，自动换到下一行。

  • 特点

    • 默认居中对齐，可以设置为左对齐或右对齐。
    • 适用于需要简单排列的场景。

  • 代码示例

    import java.awt.*;

public class Demo01FlowLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("FlowLayout Example");
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        frame.setLayout(new FlowLayout());

        frame.add(new JButton("Button 1"));
        frame.add(new JButton("Button 2"));
        frame.add(new JButton("Button 3"));
        frame.add(new JButton("Button 4"));
        frame.add(new JButton("Button 5"));

        frame.setVisible(true);
    }
}

BorderLayout

  • 简介: BorderLayout 将容器划分为五个区域：东、南、西、北和中（East, South, West, North, Center）。每个区域只能添加一个组件，未添加组件的区域保持空白。

  • 特点

    • 适用于需要在特定区域布局组件的场景。
    • 中间区域会占据所有剩余的空间。

  • 代码示例

    import java.awt.*;

public class Demo02BorderLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("BorderLayout Example");
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        frame.setLayout(new BorderLayout());

        frame.add(new JButton("North"), BorderLayout.NORTH);
        frame.add(new JButton("South"), BorderLayout.SOUTH);
        frame.add(new JButton("East"), BorderLayout.EAST);
        frame.add(new JButton("West"), BorderLayout.WEST);
        frame.add(new JButton("Center"), BorderLayout.CENTER);

        frame.setVisible(true);
    }
}

GridLayout

  • 简介: GridLayout 将容器划分为等大小的网格，每个网格中可以添加一个组件，所有组件大小相同。

  • 特点

    • 适用于需要均匀排列组件的场景。
    • 行和列的数量可以指定。

  • 代码示例

    import java.awt.*;

public class Demo03GridLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("GridLayout Example");
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        // 设置为GridLayout网格布局 行 列
        frame.setLayout(new GridLayout(2, 3)); // 2行3列的网格

        frame.add(new JButton("Button 1"));
        frame.add(new JButton("Button 2"));
        frame.add(new JButton("Button 3"));
        frame.add(new JButton("Button 4"));
        frame.add(new JButton("Button 5"));
        frame.add(new JButton("Button 6"));

        frame.setVisible(true);
    }
}

BoxLayout

  • 简介: BoxLayout 能够沿着单一轴线（X轴或Y轴）排列组件。可以创建水平（X轴）或垂直（Y轴）排列的布局。

  • 特点

    • 适用于需要沿单一方向排列组件的场景。
    • 可以通过添加垂直或水平间隔（Glue、Strut）来调整组件间距。

  • 代码示例

    import java.awt.*;

public class Demo04BoxLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("BoxLayout Example");
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        JPanel panel = new JPanel();
        // 盒子布局
        panel.setLayout(new BoxLayout(panel, BoxLayout.Y_AXIS)); // 垂直排列

        panel.add(new JButton("Button 1"));
        panel.add(Box.createVerticalStrut(10)); // 添加垂直间隔
        panel.add(new JButton("Button 2"));
        panel.add(Box.createVerticalStrut(10));
        panel.add(new JButton("Button 3"));
        panel.add(Box.createVerticalStrut(10));
        panel.add(new JButton("Button 4"));

        frame.add(panel);
        frame.setVisible(true);
    }
}

这些示例展示了布局管理器在Java Swing中的基本用法。你可以根据实际需求选择合适的布局管理器来设计你的GUI应用程序。

        // 创建JFrame实例并设置标题

        // 创建JPanel实例
        JPanel panel = new JPanel();

        // 设置窗口大小
        frame.setSize(500, 300);
        
        // 设置窗口位置
        frame.setLocationRelativeTo(null); // 居中显示
        
        // 设置窗口关闭操作
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        
        // 设置窗口可见
        frame.setVisible(true);

        // 设置JFrame获得焦点,JFrame默认没有焦点 加了才会生效
        frame.requestFocus();

        // 给登录按钮添加点击事件 方法参数是接口 传入匿名内部类
        loginButton.addActionListener(new ActionListener() {
            @Override
            public void actionPerformed(ActionEvent e) {
                System.out.println("用户点击了登录按钮" + e);
            }
        });

        // 给Frame窗口添加键盘按键，如果使用KeyListener需要重写3个抽象方法(麻烦)
        // java提供了一个KeyListener接口，但是这个接口比较麻烦，所以使用KeyAdapter类
        frame.addKeyListener(new KeyAdapter() {
            @Override
            public void keyPressed(KeyEvent e) {
                // 键盘按下时执行这个方法
                // 按键事件对象 焦点一定要在窗口上
                System.out.println("键盘按键按下："+e);
                switch(e.getKeyCode()) {
                    case KeyEvent.VK_KP_UP -> System.out.println("向上");
                    case KeyEvent.VK_KP_DOWN -> System.out.println("向下");
                    case KeyEvent.VK_KP_LEFT -> System.out.println("向左");
                    case KeyEvent.VK_KP_RIGHT -> System.out.println("向右");
                }
            }
        });
    }
}

package com.itheima.ui;

import com.itheima.MainApp;
import com.itheima.bean.User;

import javax.swing.*;
import java.awt.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class LoginWindow extends JFrame { // 定义一个继承自JFrame的类，表示主窗口

        registerButton.setBounds(180, 180, 100, 30); // 设置按钮的位置和尺寸

        loginButton.addActionListener(new ActionListener() {
            @Override
            public void actionPerformed(ActionEvent e) {
                System.out.println("按钮被点击了");
                // 1.取用户输入的用户名和密码
                String loginName = usernameField.getText();
                char[] passwordCharArray = passwordField.getPassword();
                String password = new String(passwordCharArray);
                System.out.println("用户名 = " + loginName + ", 密码 = " + password);
                // 2.如果用户名或者密码错误，提示登录失败
                if (checkLoginNameAndPassword(loginName, password)) {
                    // 3.如果用户名和密码正确，关闭登录界面，进入员工管理页面
                    // 关闭当前页面
                    dispose();
                    // 进入员工管理页面
                    new EmployeeManagerWindow();

                    // 提示登录失败
                    JOptionPane.showMessageDialog(null, "用户名或密码错误，请重新输入！");
                }
            }
        });
    }

    // 根据用户名和密码确认是否登录成功
    public boolean checkLoginNameAndPassword(String loginName, String password) {
        // 遍历用户集合
        for (int i = 0; i < MainApp.userlist.size(); i++) {
            // 获取每个用户对象
            User user = MainApp.userlist.get(i);
            // 判断用户名和密码是否正确
            if (user.getLoginName().equals(loginName) && user.getPassword().equals(password)) {
                // 如果用户名和密码相同就返回true
                return true;
            }
        }
        // 如果集合没有找到就返回false
        return false;
    }
}

        setLocationRelativeTo(null); // 居中显示窗口

        JPanel topPanel = new JPanel();
        topPanel.setLayout(new FlowLayout(FlowLayout.CENTER, 10, 5));
        topPanel.setBorder(BorderFactory.createEmptyBorder(10, 10, 10, 10));

        // 输入框

        model = new DefaultTableModel(
                new String[]{"编号", "姓名", "性别", "薪资", "部门"}, 0
        ) {
            @Override
            public boolean isCellEditable(int row, int column) {
                return false; // 设置表格单元格不可编辑
            }
        };

        // 初始化表格
        table = new JTable(model);
        table.setIntercellSpacing(new Dimension(0, 0)); // 设置单元格间距
        table.setRowHeight(30); // 设置行高
        TableColumnModel columnModel = table.getColumnModel();
        columnModel.getColumn(0).setMaxWidth(60); // 编号列宽度
        columnModel.getColumn(1).setMaxWidth(120); // 姓名列宽度

        // 滚动面板
        scrollPane = new JScrollPane(table);

        // 表头设置
        JTableHeader header = table.getTableHeader();
        header.setFont(header.getFont().deriveFont(Font.BOLD));
        header.setForeground(Color.BLACK);
        header.setBackground(Color.LIGHT_GRAY);

        // 右键菜单
        JPopupMenu popupMenu = new JPopupMenu();
        JMenuItem modifyItem = new JMenuItem("修改");
        JMenuItem deleteItem = new JMenuItem("删除");
        popupMenu.add(modifyItem);
        popupMenu.add(deleteItem);

        // 添加监听器
        table.addMouseListener(new MouseAdapter() {
            @Override
            public void mouseClicked(MouseEvent e) {
                if (e.getButton() == MouseEvent.BUTTON3 && e.getClickCount() == 1) { // 右键点击
                    int row = table.rowAtPoint(e.getPoint());
                    int col = table.columnAtPoint(e.getPoint());
                    table.setRowSelectionInterval(row, row);
                    table.changeSelection(row, col, false, false);
                    popupMenu.show(table, e.getX(), e.getY());
                }
            }
        });

        // 动作监听器
        ActionListener actionListener = e -> {
            String command = ((AbstractButton) e.getSource()).getActionCommand();
            switch (command) {
                case "修改":
                    // 修改逻辑
                    JOptionPane.showMessageDialog(EmployeeManagerWindow.this, "修改员工信息");

    public void autoMove() {

    public synchronized void move(int newDirection) {
        System.out.println("最新移动方向 newDirection = " + newDirection);

            System.out.println("不能反向移动!");
            return; // 结束方法

        // 保存最新的移动方向

        System.out.println("移动前: " + snakePointList);
        // 获取蛇头的坐标点
        Point headPoint = snakePointList.getFirst();
        // 根据最新方向移动

    public boolean isGameOver() {
        // 获取蛇头
        Point snakeHead = snakePointList.getFirst();

        // 如果蛇头坐标和墙壁坐标相同, 返回true结束游戏
        if (snakeHead.x == 0 || snakeHead.x == COLUMN_COUNT - 1 || snakeHead.y == 0 || snakeHead.y == ROW_COUNT - 1) {
            System.out.println("蛇头撞到了墙壁, 结束游戏");
            return true;
        }

        // 如果蛇头吃到自己, 返回true结束游戏
        for (int i = 1; i < snakePointList.size(); i++) {
            // 获取蛇头后面的点
            Point bodyPoint = snakePointList.get(i);
            if (snakeHead.equals(bodyPoint)) {
                System.out.println("蛇头吃到了自己, 结束游戏");
                return true;
            }
        }

        // 游戏没有结束
        return false;
    }
