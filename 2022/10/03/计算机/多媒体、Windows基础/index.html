多媒体概念

媒体一词源于英文Medium，其种类繁多。
一种是指表示信息的载体，如==文字、图形、图像声音、视频影像、动画==等，这就是多媒体计算机技术中所指的媒体;
一种是指存储信息的实体，如==纸张、半导体存储器、磁带、磁盘、光盘==等。

媒体分为5类:

感觉媒体，指通过人的感觉器官能直接感受的媒体，如视觉。
表示媒体，用于传播和表达感觉媒体的中介媒体，是信息的表示和表现形式，如JPG编码
表现媒体，是进行信息输入和输出的一类媒体，如显示器
传输媒体，是用于通信传输的信息载体，如双绞线。
存储媒体，是存放表示媒体的物理实体，如光盘

感觉媒体：文字、数字、音频；
表示媒体：编码方式；
表现媒体：外设
存储媒体：硬盘U盘
传输媒体：有线无线

离散媒体：数字
连续媒体：模拟

多媒体是指运用存储与再现技术得到的计算机中的数字信息，也就是把文本、图形、声音、影像等这些”单媒体”和计算机程序融合在一起，形成的一种人机交互式的信息交流和传播媒体。
1.载体的多样性
2.集成性
3.交互性
4.协同性
5.实时性

多媒体系统是指利用计算和技术和数字通讯网技术来处理和控制多媒体信息的系统，是由多媒体终端设备、网络设备、服务系统、多媒体软件及相关媒体数据组成的有机整体。

文本及处理技术

文本(Text)是多媒体信息最基本的麦示形式之一，可以清楚、准确地表达思想，描述概念，叙述事实等。它是人们最熟悉的信息表示方式，例如，文章，书等都以文本形式出现。文本的最显著的特点是在组织上的线性的和顺序的。文本类型：无格式文本、格式文本、超文本

色彩三要素色

亮度是光作用于人眼时所引起的明亮程度的感觉，它与被观察物体的发光强度有关。
色相是当人眼看一种或多种波长的光时所产生的彩色感觉，它反映颜色的种类，是决定颜色的基本特性。
饱和度是指颜色的纯度，也可以叫做纯度、彩度或浓度等，即掺入白光的程度，或者是指颜色的深浅程度。

HSB彩色模式
是根据日常生活中人眼的视觉特征而制定的一套色彩模式。HSB颜色模式以色相、饱和度和亮度描述颜色的基本特征。

CMY颜色模式
是采用**青(Cyan、品红或洋红(Magenta)、黄(Yellow)**3种基本颜色按一定比例合成颜色的方法。颜色的产生是来自于照射在颜料上反射回来的光线。
图像打印输出时用CMY颜色模式。

Lab颜色模式
分别用**亮度或光亮度分量(Luminosity)和两个色度分量(a、b)**来表示颜色
L表示亮度。L的值域由0到100，L=50时，相当于50%的黑。
a表示从洋红色至绿色的范围，
b表示从黄色至蓝色的范围，
a和b的值域是由+127至-128。

索引颜色模式
最多使用256种颜色，当图像被转换为索引颜色模式时，通常会构建一个调色板存放图像中的颜色并编制颜色索引。

位图模式
位图模式的图像只有黑色与白色两种像素，每个像素用1位二进制数表示,”0”表示黑色,“1”表示白色。

灰度模式
用单一色相表现图像，最多使用256级。图像中的每个像素有一个0(黑色)～255(白色)之间的亮度值。此外，灰度值也可以用黑色油墨覆盖的百分比来表示(0%表示白色，100%表示黑色)。

颜色深度
位图图像中各像素的颜色信息是用二进制数据描述的。
色彩由颜色深度决定，不是分辨率
二进制的位数就是位图图像的颜色深度。颜色深度决定了图像中可以呈现的颜色的最大数目。

分辨率:
图像分辨率(Image Resolution):指单位图像线性尺寸中所包含的像素数目，通常以**像素/英寸(pixel per inch,ppi)**为计量单位。

目前，颜色深度有1、4、8、16、24和32几种。

颜色深度为1时，表示点阵图像中各像素的颜色只有1个二进制位，可以
表示两种颜色(黑色和白色);
颜色深度为24时，表示点阵图像中各像素的颜色为3个字节(24位)，可以表示16777216种颜色;
颜色深度为32时，则用3个字节分别表示R、G、B颜色，而用另一个字
节来表示图像的其他属性(如透明度等)。
当图像的颜色深度达到或超过24时，则称这种颜色为真彩色。

  • ==编码==[编码方式]

    • 无损压缩 gif png
    • 不压缩 bmp

@@ 声音信息的数字化过程是每隔一段时间间隔在模拟声音波形上取一个幅度，称为(采样），并把得到的声音强弱用模拟电压用数字表示，称为(量化)

位图图像大小计算

==存储容量=分辨率 * 颜色深度 / 8==，颜色深度的单位是 “位 bit “，图像容量的单位是字节B
颜色种类数，如24位真彩色，颜色种类数是$2^{24}$
一张图像是1280×1024分辨率，24位真彩色，存储量是多少 MB ?

答：1280 * 1024 * 24 / 8 == 3932160字节 == 3840KB == 3.75MB [往大进位就要除1024]

常用位图的格式
  • .bmp 【不压缩】Microsoft和IBM联合开发的原始位图格式，不压缩，WinXp画图程序默认格式，是Windows操作系统中的标准图像文件格式，能够被䜶Windows应用程序支持。不采用任何压缩，所以BMP文件占用的空间很大
  • .gif 无损压缩位图，最多支持256色，分静态gif和动态gif，体积小，图像相对清晰，它在存储文件时采用LZW压缩算法，支持256色的图像，支持动画和透明。传输速度快，可将多个图像存在一个文件而形成动画效果【不能储存真彩色】
  • .jpg / jpeg JPEG联合图像专家组制定的图像压缩标准，有损压缩，目前最流行的图像格式，一种高效率的有损压缩
  • .png 无损压缩位图，支持透明效果，Win7 / Win10画图程序默认图像格式，无损压缩方式，压缩率比较高，有利于网络传输
  • .tif TIFF标签图像文件格式，无损压缩位图格式

位图与矢量图的比较

位图

  • 优点：色彩层次丰富，适合人像、风景等
  • 缺点：==缩放、旋转、变形后会失真==

矢量图

  • 优点：文件小，文件大小和分辨率无关，只与图像复杂度有关。对图形进行缩放、旋转或变形操作，不会产生锯齿状
  • 缺点：==最大的缺点是难以表现色彩层次丰富的逼真图像效果==
位图与矢量图的区别

位图叫做栅格图像，是由多个像素组成的。位图的清晰度与像素的多少有关，单位面积内像素数目越多则图像越清晰，反之图像越模糊。
矢量图又称为向量图形，是由线条和图块组成的，当对矢量图进行放大后，图形仍能保持原来的清晰度，且色彩不失真
一般情况下，==位图占用空间比矢量图大，放大后容易失真==
位图是像素点阵的描述
矢量图形是由数学公式来描述的

视频信号数字化：类似图像的方法

FPS: (Frame Per Second) FPS是图像领域中的定义，是指画面每秒传输帧数，通俗来讲就是指动画或视频的画面数。FPS是测量用于保存、显示动态视频的信息数量。每秒钟帧数越多，所显示的动作就会越流畅。通常，要避免动作不流畅的最低是30。某些计算机视频格式，每秒只能提供15帧。电影以每秒24张画面的速度播放，也就是一秒钟内在屏幕上连续投射出24张静止画面。有关动画播放速度的单位是fps，其中的f就是英文单词Frame（画面、帧），p就是Per（每），s就是Second（秒）。用中文表达就是多少帧每秒，或每秒多少帧。电影是24fps，通常简称为24帧。

视频编码: MPEG(动态图像专家组)、H.26X

计算

  • 文件大小 ： 容量=声道数 * 采样频率 * 量化位数 * 长度 / 8 其中频率单位Hz，量化单位bit，长度秒，容量单位字节B
  • 码率 ：码率 = 帧速率 * 分辨率 * 颜色深度，其中帧速率单位是fps，颜色深度单位是bit，码率单位是bps

五大件[空运存出入]

  • ==输入设备[二进制、内存]==

    • 常用输入设备：键盘、鼠标、扫描仪、数字化仪、数码摄像机、条形码阅读器、数码相机和A/D转换器

  • ==运算器[算术运算 + 逻辑运算]==

    • 由**算数逻辑运算单元(ALU)和寄存器[存储中间结果]构成用来完成FFFF算术运算和逻辑运算**
      [包含：算术逻辑单元(ALU)、 累加寄存器(AC) 、 状态条件寄存器(PSW) 、 数据缓冲器(DR)]

  • ==控制器[计算机的工作过程(发出控制计算)]==

    • 控制器是指挥和控制计算机各个部件进行工作的”神经中枢”用来控制、指挥程序和数据的输入、运行以及处理运算结果
      [包含：指令寄存器(IR)、程序计数器(PC)、地址寄存器(AR)、指令寄存器(ID)]

  • ==存储器[指令和数据]==

    • 内存[能直接跟CPU交流数据]

      • ==ROM==：Read Only Memory 只读存储器，如主板上存放BIOS程序的ROM，存在CMOS里, 有Li电池供电

        ROM-Read Only Memory只读存储器。断电后信息不丢失，如计算机启动用的BIOS芯片。存取速度很低，（较RAM而言）且不能改写。由于不能改写信息，不能升级，现已很少使用。
        主板上CMOS芯片主要用途是存储时间、日期、硬盘参数与计算机配置信息。CMOS是一种半导体，是一个RAM芯片，用来存储时间、日期、硬盘参数与计算机配置信息，因为是RAM，断电后会丢失，所以有一个专门的供电的CMOS锂电池给它供电，如果电池没电了，开机后电脑会恢复到计算机的出厂时间;BIOS是一个程序，基本输入输出系统，包括基本输入输出程序、引导程序和开机自检程序，它是固化在ROM上的，不能更改。我们说BIOS设置或者CMOS设置，实际上是通过BIOS程序来设置CMOS中的信息

        • 特点：只读不写，断电后数据不丢失
        • 改进分类：

        PROM：可编程只读存储器 [progressROM]

        EPROM：可擦写可编程只读存储器 紫外线擦除

        EEPROM：电可擦写可编程只读存储器

        Flash Memory：闪存，EEPROM，用作U盘，SSD等时间上属于外存

      • ==RAM==：Read Access Memory 随机存储器

        • 特点：可读可写，断电后数据丢失
        • 分类：

        静态随机存储器(SRAM)：Static RAM 不需刷新，速度快，但价格贵，集成度低，**Cache属于SRAM **用于制作Cache

        动态随机存储器(DRAM)：Dynamic RAM 需要不断刷新，价格低，速度比SRAM慢，常用于主存储器 用于制作内存条

        手机的 8[RAM] + 256[ROM]

      • Cache：高速缓冲存储器，简称缓存 用来解决CPU和内存速度不匹配的问题

    存储在SRAM或DRAM中的数据在断电后将全部丢失且无法恢复。RAM有两大类，一种称为静态RAM（Static RAM/SRAM），SRAM速度非常快，是目前读写最快的存储设备了，但是它也非常昂贵，所以只在要求很苛刻的地方使用，譬如CPU的一级缓冲，二级缓冲。另一种称为动态RAM（Dynamic RAM/DRAM），DRAM保留数据的时间很短，速度也比SRAM慢，不过它还是比任何的ROM都要快，但从价格上来说DRAM相比SRAM要便宜很多，计算机内存就是DRAM的。

    ∵ 答: 32KB=32K个存储单元，32K=$2^5$ * $2^{10}$ = $2^{15}$个存储单元，MDR的位数是15，也就是000 0000 0000 0000 - 111 11111 1111 1111，则内存存储地址范围为0000H-7FFFH，存储单元个数是7FFF-0+1=8000H，=8 * $16^3$个存储单元，

指令

指令是指示计算机执行某种操作的命令，由一串二进制码组成

==指令包括 操作码 和 操作数(地址码) 两部分== 操作码规定了操作的类型，进行什么样的操作；地址码规定了要操作的数据(操作对象)存放在什么地址中，以及操作结果存放到哪个地址中去。

程序：为了完成某个任务而设计的一组有序的指令的集合

指令系统

  • 所有指令的集合成为计算机指令系统
  • 复杂指令系统(CISC)
  • 精简指令系统(RISC) 格式统一、种类少、寻址方式少、处理速度快

计算机工作过程

  • 取指令-分析指令-执行指令，指令计数器加1
  • 几乎所有的冯·诺依曼型计算机的CPU，其工作都可以分为5个阶段：取指令、指令译码、执行指令、访存取数、结果写回

==微机的主要性能指标==[主子内部存在运算]

  • ==主频==/时钟频率：计算机CPU的时钟频率，主频的单位是兆赫兹（MHz）或吉赫兹（GHz）

即时钟频率，是指计算机CPU在单位时间内发出的脉冲数，它在很大程度上决定了计算机的运算速度。

主频的单位是赫兹(Hz) MHz、GHz

  • ==字长==：计算机CPU能够直接处理的二进制数据的位数

字长是指计算机的运算部件能同时处理的二进制数据的位数；字长取决于数据总线的宽度；

​ 字长一般为字节的整数倍，如8位、16位、32位，现在主流都是64位

  • 功能

字长越长，计算机处理信息的效率就越高，计算机内部所存储的数值精度就越高，计算机所能识别的指令数量就越多，功能也就越强

字长决定了指令直接寻址的能力

  • 内核数

CPU内核数指CPU内执行指令的运算器和控制器的数量

多核CPU 是一个CPU有多个核心

  • ==存储容量==：分内存容量和外存容量

内存容量越大，存储的程序和数据越多，处理能力和速度越强

  • 存取周期：CPU从内存中存取数据所需的时间，存取周期越短，运算速度越快

存取时间：内存储器完成一次读(取)或写(存)操作所需的时间成为存储器的存储时间或访问时间

存取周期：连续两次独立的读(或写)所需的最短时间称为存取周期

  • ==运算速度==：一般用百万次/秒（MIPS）来描述

单位时间内执行的计算机指令数。单位：MIPS和BIPS

操作系统概述

操作系统的作用

  • OS作为用户与计算机硬件系统之间的接口
  • OS管理计算机软硬件资源
  • OS实现了对计算机资源的抽象

操作系统的特征[一不虚拟就并发着共享]

  • 并发性：同一时间段内操作系统允许多个程序运行；两个或两个以上的运行程序在同一时间间隔段内同时执行
  • 共享性：计算机的多个程序可以共享各种硬件资源和软件资源；操作系统中的资源可被多个并发执行的程序所使用
    • 互斥共享方式：打印机
    • 同时访问方式：磁盘
  • 虚拟性：时[间]分复用技术、空[间]分复用技术；指将一个屋里实体映射为若干逻辑实体(虚拟处理机，虚拟存储器)
  • 异步性：随机性；指的是操作系统能针对各种随机事件，键盘输入、错误产生等提供支持，而这些事件的发生是随机的。

操作系统的功能[五大功能模块 ]

  • 处理机管理

    • 处理终端事件

    • 处理器调度

    • 作业处理方式(CPU)

      • 批处理
      • ==分时处理==：分时操作系统利用PCU的空余时间处理成批的作业
      • ==实时处理==：实时操作系统是对外来信号及时做出反应的操作系统

    • 进程：作业管理的功能是把用户的作业装入内存并投入运行，一旦作业进入内存，就成为进程[它是一段程序的执行过程，是系统进行资源分配和调度的基本单位；进程是程序的一次执行]

      • ==特征：**动态性、并发性、独立性、异步性**===
      • 基本状态：就绪 -> 执行 -> 阻塞
      • ==线程==：是操作系统能够进行运算调度的最小单位。它被包含在进程之中，是进程中的实际运作单位。

      软件是由文档完成任务+程序完成功能 程序是静态文件 进程是动态 程序有创建进程的功能
      进程是描述并发的 CPU资源按进程分配 以进程为单位进行分配 进程=多个线程+数据

      • 多线程技术：多线程就是把操作系统中的并发执行机制原理运用在一个程序中，把一个程序划分成若干个子任务，多个子任务并发执行，每个任务就是一个线程
      • 用户态：应用软件的权限
      • 内核态：系统软件的权限

      用户态和内核态的切换是系统的调度

  • 存储管理

    • 主要任务：管理存储器资源，为多道程序运行提供有力支撑
    • 主要功能：存储分配、存储共享、存储包含和存储扩充[虚拟存储技术]

  • 设备管理：管理各类外围设备、设备分配、设备处理

  • 文件管理

    • 主要任务是对用户文件和系统文件进行管理以方便用户使用，并确保文件的安全性
    • 包括对文件存储空间的管理、目录管理、文件的读写管理，以及文件的共享与保护

  • 作业管理：用户需要计算机完成某项任务时要求计算机所做工作的集合称为作业

一个物理磁盘分成几个逻辑磁盘(分区)，每个分区由一组连续的扇区组成。

扇区：Windows系统中，硬盘不是一次读写一个字节而是一次读写一个扇区(512个字节)，扇区是硬盘存储信息，物理上读写的最小单位/基本单位

簇：系统读写文件的基本单位，文件系统管理最小的磁盘单位，一般为2的n次方个扇区(由文件系统决定)

文件：记录在磁盘上的一组相关信息的集合，文件是操作系统管理用户信息的基本单位

多媒体技术

多媒体技术 是 计算机 与 媒体技术 相结合的产物
【重点】多媒体技术的概念：是指通过计算机对 文字、数字、图形、图像、音频、视频、动画 等多种媒体信息进行综合处理和管理，使用户可以通过多种感官与计算机进行实时信息交互的技术。
【说明】

1.媒体 分为 媒质 和 媒介 两类
(1) 媒质：指保存信息的存储器，如 硬盘、U盘
(2) 媒介：指信息的载体，如 文字、数字、…【最主要的形式是文字】

2.==媒体分为 五大类==
(1) 感觉媒体：指 听觉、视觉 等人体感官
(2) 表示媒体：指 信息在计算机中的 编码形式
如，西文字符采用ASCII码、整数采用补码、汉字采用机内码、静态图像采用 JPEG(静态图像编码)、视频采用MPEG(动态图像编码)
(3) 表现媒体(显示媒体)：指 外设，如 显示器、键盘、鼠标、打印机
(4) 存储媒体：指保存媒体的存储器，如 硬盘、U盘
(5) 传输媒体：指网络中有 有线介质 和 无线介质
【填空】多媒体技术[浇树鸡屎多] 特点：集成性、交互性、实时性、多样性、数字化、控制性、非线性、信息使用的方便性、信息结构的动态性。
① 集成性：将多种媒体形式，如 文字和声音，集成为一条信息传播
② 交互性：为用户 提供控制和使用信息 的手段
③ 实时性：图像、文字、视频 等连续播放，与 现场同时，具有 实时性 如 主播带货
④ 数字化：指所有媒体采用 二进制 的数字形式
【考点】数字化 优点：易于 压缩、加密、抗扰乱，易于 加工、传输、成本低
⑤ 多样性：传播信息时采用多种媒体形式，如 同时采用文字、音频、视频等媒体形式继承后传播

音频 数字化

【填空】音频 数字化三个过程：采样、量化、编码
“采样”：每隔固定的时间间隔，对模拟信号(单位：Hz)进行取值
【考点】采用频率越高：音质越好、但占用的存储空间越大
例：采用 44.1KHz、48KHz
“量化”：将采样得到的模拟信号，转换为二进制数字信号(A/D过程)，量化位(bit)
【考点】量化数位越大：音质越好，但需要更大的存储空间
“编码”：将量化后的数据，按一定格式记录，即：编码方式
JPEG：联合图像专家组标准，是静态图像压缩标准，如 .jpg .jpeg
MPEG：动态图像专家组标准，是视频图像压缩标准，
分为：
MPEG-1：mp3、wma 音频
MPEG-2：mp4、wmv视频
【选择题】计算机处理多媒体信息，从时效性
分为：静态媒体(文字、数字、图像)，动态媒体(音频、视频、动画)
【计算题】数字音频技术标准(影响音频数字化质量的因素)
① 采样频率
② 量化位数
③ 声道数：每次采样生成的波形文件(声音文件)的个数
单声道(生成一个声音文件)、双声道(立体声，生成两个文件)
④ 时长
例，一段1分钟的立体声、采样频率44.1KHz, 采样精度16位，数字化后不做任何压缩

60 × 2 × 44.1 × 1000 × 16 = 84672000
84672000 ÷ 8 ÷ 1024 = 10335kb 约为 10.09Mb

图像技术

【考点】图形 与 图像 的区别
图形 是 矢量图，是计算机绘制的直线、曲线、点 等组成的形状，是计算机程序算出来的。
【考点】PDF->缩放、旋转、移动 不会变形(不失真)，占用的 存储空间小；但：色彩逼真度差(与自然界颜色相差甚远)

图像 是 位图(自拍照) 利用 照相机、摄像机、扫描仪 等捕捉的自然界画面
【特点】缩放、旋转 会变形，占用的存储空间大；但 图像色彩丰富、逼真

图像数字化 三个过程：采样、量化、编码
【复习】RGB 三原色 一个颜色3个字节 占用 3三种颜色 * 8bit = 24bit = 3字节

【例题】图像采样频率是 640*480(px 像素)，颜色位数(颜色深度)是 24位，求图像占用的存储空间
【说明】颜色位数(颜色深度)是指 每个像素中所携带的表示颜色的位数 存储空间=640 * 480px * 24bit / 8 = 0.9Mb
【总结】影响 图像数字化的因素有：图像分辨率、颜色位数
JPEG图像格式可以选择不同的压缩比对图像进行压缩，由于JPEG格式在压缩时将丢失部分信息，且不能还原，不适合方法观看。输出成印刷品时质量也受影响。压缩比越大，图像的边缘失真也越明显
PNG格式是一种新型的网络图像格式，压缩比例高且不失真(无损压缩)，有利于网络传播。支持透明图像的制作
