图像信息在计算机中的表示

位图

图像信息的数字化
  • 采样

图像分辨率，位图由n * m的像素矩阵组成，每个像素点只保存颜色信息，如1920 * 1080 (1080p) ， 2560 * 1440(2K) 、3840 * 2160 (4K）等不同的分辨率

  • 量化

颜色深度，每个像素点使用多少位二进制保存颜色信息，有8位色、16位色、24位色(真彩色)、32位色(24位色+8位alpha通道)

  • 编码
    • 有损压缩 JPEG
    • 无损压缩 gif png
    • 不压缩 bmp
位图图像大小计算

存储容量=分辨率 * 颜色深度 / 8，颜色深度的单位是 “位 bit “，图像容量的单位是字节B
颜色种类数，如24位真彩色，颜色种类数是2的24次方
一张图像是1280×1024分辨率，24位真彩色，存储量是多少 MB ?

答：1280 * 1024 * 24 / 8 == 3932160字节 == 3840KB == 3.75MB [往大进位就要除1024]

常用位图的格式
  • .bmp Microsoft和IBM联合开发的原始位图格式，不压缩，WinXp画图程序默认格式
  • .gif 无损压缩位图，最多支持256色，分静态gif和动态gif，体积小，图像相对清晰
  • .jpg / jpeg JPEG联合图像专家组制定的图像压缩标准，有损压缩，目前最流行的图像格式
  • .png 无损压缩位图，支持透明效果，Win7 / Win10画图程序默认图像格式
  • .tif TIFF标签图像文件格式，无损压缩位图格式

矢量图

矢量图使用直线和曲线描述图形，通过数学公式计算获得，常用于图案、标志、VI、文字等设计

常见矢量图格式
  • .swf
  • AI、CAD、CDR等 <==> .ai .dwg .cdr
  • TrueType字体

位图与矢量图的比较

位图

  • 优点：色彩层次丰富，适合人像、风景等
  • 缺点：缩放、旋转、变形后会失真

矢量图

  • 优点：文件小，文件大小和分辨率无关，只与图像复杂度有关。对图形进行缩放、旋转或变形操作，不会产生锯齿状
  • 缺点：最大的缺点是难以表现色彩层次丰富的逼真图像效果

音频信息在计算机中的表示

波形数字音频

音频信息处理的流程
  • A/D转换[采样]，模拟音频的数字化【影响数字化质量的主要原因】

  • D/A转换，数字音频模拟化输出
计算
  • 波形音频文件容量的计算 容量=声道数 * 采样频率 * 量化位数 * 长度 / 8 其中频率单位Hz，量化单位bit，长度秒，容量单位字节B
  • 波形音频文件码率的计算 码率=声道数 * 采样频率 * 量化位数 码率的单位是bps
常见波形文件格式
  • .wav 微软和IBM共同开发的PC标准音频格式，未压缩，声音达到CD音质，码率约为1.4Mb/s，Windows XP录音机默认音频格式
  • .mp3 有损压缩，最常用 互联网、MP3音乐
  • .wma 微软公司的有损压缩，压缩比高于MP3，Win7录音机默认格式
  • .m4a 苹果公司的无损压缩
  • .flac 无损压缩，高品质数字音乐
  • .ape 无损压缩音频格式
MIDI格式
  • MIDI乐器数字接口，垫桌子音乐制造商们建立的通信标准
  • MIDI传输的不是声音信号，而是音符、控制参数等指令，它指示MIDI设备要做什么，怎么做; 如演奏哪个音符，音量多大等
  • .mid 或 .midi

视频信息在计算机中的表示

视频信号数字化：类似图像的方法

FPS: (Frame Per Second) FPS是图像领域中的定义，是指画面每秒传输帧数，通俗来讲就是指动画或视频的画面数。FPS是测量用于保存、显示动态视频的信息数量。每秒钟帧数越多，所显示的动作就会越流畅。通常，要避免动作不流畅的最低是30。某些计算机视频格式，每秒只能提供15帧。电影以每秒24张画面的速度播放，也就是一秒钟内在屏幕上连续投射出24张静止画面。有关动画播放速度的单位是fps，其中的f就是英文单词Frame（画面、帧），p就是Per（每），s就是Second（秒）。用中文表达就是多少帧每秒，或每秒多少帧。电影是24fps，通常简称为24帧。

视频编码: MPEG(动态图像专家组)、H.26X

计算

  • 文件大小 ： 容量=声道数 * 采样频率 * 量化位数 * 长度 / 8 其中频率单位Hz，量化单位bit，长度秒，容量单位字节B
  • 码率 ：码率 = 帧速率 * 分辨率 * 颜色深度，其中帧速率单位是fps，颜色深度单位是bit，码率单位是bps

计算机系统

早期的冯 · 诺依曼机可以存储程序

冯诺依曼机的特点:

1.计算机由运算器、控制器、存储器、输入设备和输出设备组成

2.指令和数据以同等地位存放于存储器内，并可按地址寻访

3.指令和数据均用二进制数表示

4.指令由操作码和地址码组成，操作码用来表示操作的性质，地址码用来表示操作数在存储器中的位置

5.存储程序

6.以运算器为中心

原来的计算机以：*运算器*为中心

现在的计算机以：存储器为中心

硬件 = 主机 + 外设

主机 = CPU (CPU = 运算器 + 控制器) + 内存

外设 = 输入设备 + 输出设备 + 外存

存储器 = 内存 + 外存

五大件

  • 输入设备

    • 常用输入设备：键盘、鼠标、扫描仪、数字化仪、数码摄像机、条形码阅读器、数码相机和A/D转换器

  • 运算器

    • 由算数逻辑运算单元(ALU)和寄存器[存储中间结果]构成用来完成算术运算和逻辑运算

  • 控制器

    • 控制器是指挥和控制计算机各个部件进行工作的”神经中枢”用来控制、指挥程序和数据的输入、运行以及处理运算结果

  • 存储器

    • 内存[能直接跟CPU交流数据]

      • ROM：Read Only Memory 只读存储器，如主板上存放BIOS程序的ROM，存在CMOS里，有Li电池供电

        • 特点：只读不写，断电后数据不丢失
        • 改进分类：

        PROM：可编程只读存储器 [progressROM]

        EPROM：可擦写可编程只读存储器 紫外线擦除

        EEPROM：电可擦写可编程只读存储器

        Flash Memory：闪存，EEPROM，用作U盘，SSD等时间上属于外存

      • RAM：Read Access Memory 随机存储器，微机中的内存一般指RAM，存CMOS的也是RAM[微机中的内存一般指随机存储器]

        • 特点：可读可写，断电后数据丢失
        • 分类：

        静态随机存储器(SRAM)：Static RAM 不需刷新，速度快，但价格贵，集成度低，Cache属于SRAM

        动态随机存储器(DRAM)：Dynamic RAM 需要不断刷新，价格低，速度比SRAM慢，常用于主存储器

        手机的 8[RAM] + 256[ROM]

      • Cache：高速缓冲存储器，简称缓存 用来解决CPU和内存速度不匹配的问题

      CPU → Cache → 内存 → 外存，因为CPU速度太快了，内存赶不上它的速度，就需要Cache来缓冲

      8 bit = 1 B

      • 内存单元与内存地址：

      内存单元：1个内存单元就是一个字节B

      内存地址：一串二进制数字，用来唯一表示每一个内存单元的编号，内存地址的范围由地址总线决定

    • 外存：外存是主机的外部设备，用来存储大量的不参与运算或处理的数据和程序，如硬盘、光盘、U盘、移动硬盘等

      • 硬盘

      机械硬盘(HDD)：存储介质 磁介质；参数：磁头数、柱面数、扇区(硬盘的基本读写单位，大小是512B)；

      硬盘容量 = 磁头数 * 柱面数 * 扇区数 * 512B；性能指标：存储容量、转速、访问时间、传输速率、缓存等；

      接口：SATA、IDE、SCSI、光纤通道

      固态硬盘(SSD)：存储介质(Flash Memory闪存) 接口 SATA等

      • 光盘

      读取速度 CD的1×是150KB/s，DVD的1×是1358KB/s

      CD/VCD 容量：650MB CD-ROM(只读光盘)、CD-R(一次性写入、永久读的)、CD-RW(可重复擦写光盘)

      DVD 容量： 4.7GB DVD-ROM、DVD-R、DVD-RW

    存储在SRAM或DRAM中的数据在断电后将全部丢失且无法恢复。RAM有两大类，一种称为静态RAM（Static RAM/SRAM），SRAM速度非常快，是目前读写最快的存储设备了，但是它也非常昂贵，所以只在要求很苛刻的地方使用，譬如CPU的一级缓冲，二级缓冲。另一种称为动态RAM（Dynamic RAM/DRAM），DRAM保留数据的时间很短，速度也比SRAM慢，不过它还是比任何的ROM都要快，但从价格上来说DRAM相比SRAM要便宜很多，计算机内存就是DRAM的。

    存储器的性能指标：容量、速度

    存储容量是指存储器可以容纳的二进制信息量

    内存存储 = 存储单元个数 * 存储单元字长 (bit)

    @@ 计算机中地址的概念是内存各存储单元的编号。现有一个32KB的存储器，用十六进制数对它进行编码，则编号从0000H到(7FFF)H

    ∵ 答: 32KB=32K个存储单元，32K=$2^5$ * $2^{10}$ = $2^{15}$个存储单元，MDR的位数是15，也就是000 0000 0000 0000 - 111 11111 1111 1111，则内存存储地址范围为0000H-7FFFH，存储单元个数是7FFF-0+1=8000H，=8 * $16^3$个存储单元，
    =$2^3$ * $2^{12}$ = $2^{15}$个存储单元=32K。如果存储单元字长是8bit，32K * 8bit = 32KB。

    一个字节就是一个存储单元！

指令

指令是指示计算机执行某种操作的命令，由一串二进制码组成

指令包括操作码和操作数(地址码)两部分 操作码规定了操作的类型，进行什么样的操作；地址码规定了要操作的数据(操作对象)存放在什么地址中，以及操作结果存放到哪个地址中去。

程序：为了完成某个任务而设计的一组有序的指令的集合

指令系统

  • 所有指令的集合成为计算机指令系统
  • 复杂指令系统(CISC)
  • 精简指令系统(RISC) 格式统一、种类少、寻址方式少、处理速度快

计算机工作过程

  • 取指令-分析指令-执行指令，指令计数器加1
  • 几乎所有的冯·诺依曼型计算机的CPU，其工作都可以分为5个阶段：取指令、指令译码、执行指令、访存取数、结果写回

微机的主要性能指标

  • 主频

即时钟频率，是指计算机CPU在单位时间内发出的脉冲数，它在很大程度上决定了计算机的运算速度。

主频的单位是赫兹(Hz) MHz、GHz

  • 字长

字长是指计算机的运算部件能同时处理的二进制数据的位数；字长取决于数据总线的宽度；

字长一般为字节的整数倍，如8位、16位、32位，现在主流都是64位

  • 功能

字长越长，计算机处理信息的效率就越高，计算机内部所存储的数值精度就越高，计算机所能识别的指令数量就越多，功能也就越强

字长决定了指令直接寻址的能力

  • 内核数

CPU内核数指CPU内执行指令的运算器和控制器的数量

多核CPU 是一个CPU有多个核心

  • 内存容量

内存容量越大，存储的程序和数据越多，处理能力和速度越强

  • 存取周期

存取时间：内存储器完成一次读(取)或写(存)操作所需的时间成为存储器的存储时间或访问时间

存取周期：连续两次独立的读(或写)所需的最短时间称为存取周期

  • 运算速度

单位时间内执行的计算机指令数。单位：MIPS和BIPS

    • 常见总线标准(接口)：CPU跟内存条之间的为局部总线、内存条跟LCD显示屏信息的传递为设备总线。

        • PS/2：鼠标、键盘接口，紫色接键盘、绿色接鼠标

        • VGA：不支持热插拔，不支持音频传输

        • HDMI：可以同时发送音频和视频信号；可用于机顶盒、DVD播放机、个人计算机、电视、游戏主机、数字音箱与电视机设备

  • 主板：是一块带有各种插口的大型印刷电路板(PCB)

    • 芯片组：主板最重要的部件之一是芯片组，芯片组是主板的灵魂
      • 北桥芯片：提供对CPU类型和主频的支持
      • 南桥芯片：提供了对I/O的支持

      • 主板

操作系统概述

操作系统的作用

  • OS作为用户与计算机硬件系统之间的接口
  • OS管理计算机软硬件资源
  • OS实现了对计算机资源的抽象

操作系统的特征

  • 并发性：程序并发执行
  • 共享性：系统中的资源可供内存中的多个并发执行的进程共同使用
    • 互斥共享方式：打印机
    • 同时访问方式：磁盘
  • 虚拟性：时[间]分复用技术、空[间]分复用技术
  • 异步性：进程是以人们不可预知的速度向前推进的

操作系统的功能[五大功能模块 ]

  • 处理机管理

    • 处理终端事件

    • 处理器调度

    • 作业处理方式(CPU)

      • 批处理
      • 分时处理
      • 实时处理

    • 进程：作业管理的功能是把用户的作业装入内存并投入运行，一旦作业进入内存，就成为进程[它是一段程序的执行过程，是系统进行资源分配和调度的基本单位；进程是程序的一次执行]

      • 特征：**动态性、并发性、独立性、异步性**
      • 基本状态：就绪 -> 执行 -> 阻塞
      • 线程：是操作系统能够进行运算调度的最小单位。它被包含在进程之中，是进程中的实际运作单位。
      • 多线程技术：多线程就是把操作系统中的并发执行机制原理运用在一个程序中，把一个程序划分成若干个子任务，多个子任务并发执行，每个任务就是一个线程
      • 用户态：应用软件的权限
      • 内核态：系统软件的权限

      用户态和内核态的切换是系统的调度

  • 存储管理

    • 主要任务：管理存储器资源，为多道程序运行提供有力支撑
    • 主要功能：存储分配、存储共享、存储包含和存储扩充[虚拟存储技术]

  • 设备管理：管理各类外围设备、设备分配、设备处理

  • 文件管理

    • 主要任务是对用户文件和系统文件进行管理以方便用户使用，并确保文件的安全性
    • 包括对文件存储空间的管理、目录管理、文件的读写管理，以及文件的共享与保护

  • 作业管理：用户需要计算机完成某项任务时要求计算机所做工作的集合称为作业

磁盘的分区

Windows 7 启动过程：BIOS–MBR(主引导扇区) – Bootmgr(启动管理器) – BCD(引导配置数据) –Winload.exe(加载程序) – 内核加载

Windows中，一个分区称为一个逻辑盘，各有自己的盘符，A盘B盘是以前软盘使用[虽然如今被淘汰，但是在历史上做出过贡献]，所以都是从C盘D盘开始。
一个物理磁盘分成几个逻辑磁盘(分区)，每个分区由一组连续的扇区组成。

扇区：Windows系统中，硬盘不是一次读写一个字节而是一次读写一个扇区(512个字节)，扇区是硬盘存储信息，物理上读写的最小单位/基本单位

簇：系统读写文件的基本单位，文件系统管理最小的磁盘单位，一般为2的n次方个扇区(由文件系统决定)

文件：记录在磁盘上的一组相关信息的集合，文件是操作系统管理用户信息的基本单位

概念：格式化(format)是指对磁盘或磁盘中的分区( partition )进行初始化的一种操作，这种操作通常会导致现有的磁盘或分区中所有的文件被清除。

**虚拟内存**：虚拟内存是计算机系统内存管理的一种技术。虚拟内存别称虚拟存储器（Virtual Memory )。电脑中所运行的程序均需经由内存执行，若执行的程序占用内存很大或很多，则会导致内存消耗殆尽。为解决该问题，Windows中运用了虚拟内存技术，即匀出一部分硬盘空间来充当内存使用。当内存耗尽时，电脑就会自动调用硬盘来充当内存，以缓解内存的紧张。[用户使用内存 > RAM]

回收站

还原：回收站是一个特殊的文件夹，默认在每个硬盘分区根目录下的RECYCLER文件夹中，而且是隐藏的。当你将文件删除并移到回收站后，实质上就是把它放到了这个文件夹，仍然占用磁盘的空间。只有在回收站里删除它或清空回收站才能使文件真正地删除，为电脑获得更多的磁盘空间。

永久删除：shift + delete；设置删除不放入回收站属性；命令提示符删除[ del 6666.txt ]

格式化(format)

低级格式化(物理格式化)[重新装修]：低级格式化是将硬盘划分出柱面和磁道，再将磁道划分为若干个扇区，对逻辑坏道进行修复，对物理坏道进行屏蔽隐藏。

高级格式化(逻辑格式化)[重新分配房间格局]：根据用户选定的文件系统（如FAT32、NTFS、exFAT等），在磁盘的特定区域写入特定数据，以达到初始化磁盘或磁盘分区、清除原磁盘或磁盘分区中所有文件的一个操作（扫描磁盘并检查坏道)

快速格式化：仅清除内容，不扫描磁盘检查坏道；只适用于曾经格式化过的磁盘

磁盘也属于文件夹，可以通过右键属性 - 共享 - 高级属性中设置共享文件夹

盘符可以更改的 E: -> F: [在计算机管理找存储，右键磁盘，更改驱动器号和路径]
