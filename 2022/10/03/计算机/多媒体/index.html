图像信息在计算机中的表示

位图

图像信息的数字化
  • 采样

图像分辨率，位图由n * m的像素矩阵组成，每个像素点只保存颜色信息，如1920 * 1080 (1080p) ， 2560 * 1440(2K) 、3840 * 2160 (4K）等不同的分辨率

  • 量化

颜色深度，每个像素点使用多少位二进制保存颜色信息，有8位色、16位色、24位色(真彩色)、32位色(24位色+8位alpha通道)

  • 编码
    • 有损压缩 JPEG
    • 无损压缩 gif png
    • 不压缩 bmp
位图图像大小计算

存储容量=分辨率 * 颜色深度 / 8，颜色深度的单位是 “位 bit “，图像容量的单位是字节B
颜色种类数，如24位真彩色，颜色种类数是2的24次方
一张图像是1280×1024分辨率，24位真彩色，存储量是多少 MB ?

答：1280 * 1024 * 24 / 8 == 3932160字节 == 3840KB == 3.75MB [往大进位就要除1024]

常用位图的格式
  • .bmp Microsoft和IBM联合开发的原始位图格式，不压缩，WinXp画图程序默认格式
  • .gif 无损压缩位图，最多支持256色，分静态gif和动态gif，体积小，图像相对清晰
  • .jpg / jpeg JPEG联合图像专家组制定的图像压缩标准，有损压缩，目前最流行的图像格式
  • .png 无损压缩位图，支持透明效果，Win7 / Win10画图程序默认图像格式
  • .tif TIFF标签图像文件格式，无损压缩位图格式

矢量图

矢量图使用直线和曲线描述图形，通过数学公式计算获得，常用于图案、标志、VI、文字等设计

常见矢量图格式
  • .swf
  • AI、CAD、CDR等 <==> .ai .dwg .cdr
  • TrueType字体

位图与矢量图的比较

位图

  • 优点：色彩层次丰富，适合人像、风景等
  • 缺点：缩放、旋转、变形后会失真

矢量图

  • 优点：文件小，文件大小和分辨率无关，只与图像复杂度有关。对图形进行缩放、旋转或变形操作，不会产生锯齿状
  • 缺点：最大的缺点是难以表现色彩层次丰富的逼真图像效果

音频信息在计算机中的表示

波形数字音频

音频信息处理的流程
  • A/D转换[采样]，模拟音频的数字化【影响数字化质量的主要原因】

  • D/A转换，数字音频模拟化输出
计算
  • 波形音频文件容量的计算 容量=声道数 * 采样频率 * 量化位数 * 长度 / 8 其中频率单位Hz，量化单位bit，长度秒，容量单位字节B
  • 波形音频文件码率的计算 码率=声道数 * 采样频率 * 量化位数 码率的单位是bps
常见波形文件格式
  • .wav 微软和IBM共同开发的PC标准音频格式，未压缩，声音达到CD音质，码率约为1.4Mb/s，Windows XP录音机默认音频格式
  • .mp3 有损压缩，最常用 互联网、MP3音乐
  • .wma 微软公司的有损压缩，压缩比高于MP3，Win7录音机默认格式
  • .m4a 苹果公司的无损压缩
  • .flac 无损压缩，高品质数字音乐
  • .ape 无损压缩音频格式
MIDI格式
  • MIDI乐器数字接口，垫桌子音乐制造商们建立的通信标准
  • MIDI传输的不是声音信号，而是音符、控制参数等指令，它指示MIDI设备要做什么，怎么做; 如演奏哪个音符，音量多大等
  • .mid 或 .midi

视频信息在计算机中的表示

视频信号数字化：类似图像的方法

FPS: (Frame Per Second) FPS是图像领域中的定义，是指画面每秒传输帧数，通俗来讲就是指动画或视频的画面数。FPS是测量用于保存、显示动态视频的信息数量。每秒钟帧数越多，所显示的动作就会越流畅。通常，要避免动作不流畅的最低是30。某些计算机视频格式，每秒只能提供15帧。电影以每秒24张画面的速度播放，也就是一秒钟内在屏幕上连续投射出24张静止画面。有关动画播放速度的单位是fps，其中的f就是英文单词Frame（画面、帧），p就是Per（每），s就是Second（秒）。用中文表达就是多少帧每秒，或每秒多少帧。电影是24fps，通常简称为24帧。

视频编码: MPEG(动态图像专家组)、H.26X

计算

  • 文件大小 ： 容量=声道数 * 采样频率 * 量化位数 * 长度 / 8 其中频率单位Hz，量化单位bit，长度秒，容量单位字节B
  • 码率 ：码率 = 帧速率 * 分辨率 * 颜色深度，其中帧速率单位是fps，颜色深度单位是bit，码率单位是bps

计算机系统

早期的冯 · 诺依曼机可以存储程序

冯诺依曼机的特点:

1.计算机由运算器、控制器、存储器、输入设备和输出设备组成

2.指令和数据以同等地位存放于存储器内，并可按地址寻访

3.指令和数据均用二进制数表示

4.指令由操作码和地址码组成，操作码用来表示操作的性质，地址码用来表示操作数在存储器中的位置

5.存储程序

6.以运算器为中心

原来的计算机以：*运算器*为中心

现在的计算机以：存储器为中心

硬件 = 主机 + 外设

主机 = CPU (CPU = 运算器 + 控制器) + 内存

外设 = 输入设备 + 输出设备 + 外存

存储器 = 内存 + 外存

五大件

  • 输入设备

    • 常用输入设备：键盘、鼠标、扫描仪、数字化仪、数码摄像机、条形码阅读器、数码相机和A/D转换器

  • 运算器

    • 由**算数逻辑运算单元(ALU)**和寄存器[存储中间结果]构成 用来完成算术运算和逻辑运算

  • 控制器

    • 控制器是指挥和控制计算机各个部件进行工作的”神经中枢” 用来控制、指挥程序和数据的输入、运行以及处理运算结果

  • 存储器

    • 内存[能直接跟CPU交流数据]

      • ROM：Read Only Memory 只读存储器，如主板上存放BIOS程序的ROM，存在CMOS里，有Li电池供电

        • 特点：只读不写，断电后数据不丢失
        • 改进分类：

        PROM：可编程只读存储器 [progressROM]

        EPROM：可擦写可编程只读存储器 紫外线擦除

        EEPROM：电可擦写可编程只读存储器

        Flash Memory：闪存，EEPROM，用作U盘，SSD等时间上属于外存

      • RAM：Read Access Memory 随机存储器，微机中的内存一般指RAM，存CMOS的也是RAM

        • 特点：可读可写，断电后数据丢失
        • 分类：

        静态随机存储器(SRAM)：Static RAM 不需刷新，速度快，但价格贵，集成度低，Cache属于SRAM

        动态随机存储器(DRAM)：Dynamic RAM 需要不断刷新，价格低，速度比SRAM慢，常用于主存储器

        手机的 8[RAM] + 256[ROM]

      • Cache：高速缓冲存储器，简称缓存 用来解决CPU和内存速度不匹配的问题

      CPU → Cache → 内存 → 外存，因为CPU速度太快了，内存赶不上它的速度，就需要Cache来缓冲

      8 bit = 1 B

      • 内存单元与内存地址：

      内存单元：1个内存单元就是一个字节B

      内存地址：一串二进制数字，用来唯一表示每一个内存单元的编号，内存地址的范围由地址总线决定

    • 外存：外存是主机的外部设备，用来存储大量的不参与运算或处理的数据和程序，如硬盘、光盘、U盘、移动硬盘等

      • 硬盘

      机械硬盘(HDD)：存储介质 磁介质；参数：磁头数、柱面数、扇区(硬盘的基本读写单位，大小是512B)；

      硬盘容量 = 磁头数 * 柱面数 * 扇区数 * 512B；性能指标：存储容量、转速、访问时间、传输速率、缓存等；

      接口：SATA、IDE、SCSI、光纤通道

      固态硬盘(SSD)：存储介质(Flash Memory闪存) 接口 SATA等

      • 光盘

      读取速度 CD的1×是150KB/s，DVD的1×是1358KB/s

      CD/VCD 容量：650MB CD-ROM(只读光盘)、CD-R(一次性写入、永久读的)、CD-RW(可重复擦写光盘)

      DVD 容量： 4.7GB DVD-ROM、DVD-R、DVD-RW

    存储器的性能指标：容量、速度

    存储容量是指存储器可以容纳的二进制信息量

    内存存储 = 存储单元个数 * 存储单元字长 (bit)

    @@ 计算机中地址的概念是内存各存储单元的编号。现有一个32KB的存储器，用十六进制数对它进行编码，则编号从0000H到(7FFF)H

    ∵ 答: 32KB=32K个存储单元，32K=$2^5$ * $2^{10}$ = $2^{15}$个存储单元，MDR的位数是15，也就是000 0000 0000 0000 - 111 11111 1111 1111，则内存存储地址范围为0000H-7FFFH，存储单元个数是7FFF-0+1=8000H，=8 * $16^3$个存储单元，
    =$2^3$ * $2^{12}$ = $2^{15}$个存储单元=32K。如果存储单元字长是8bit，32K * 8bit = 32KB。

    一个字节就是一个存储单元！

    • 内存和外存的区别

    内存和外存的根本区别是，是否可以被CPU直接寻址。可以被CPU寻址的，那就是内存
    内存比外存速度快，内存比外存价格贵 外存比内存容量大、价格便宜，速度比内存慢
    打开一个文件或程序，实际是把程序或数据从外存调入内存，供CPU使用(又叫读盘)

    写盘 保存到硬盘，关闭

    三级存储体系示意图 CPU-Cache-内存-外存

    • 三级存储体系 CPU - Cache[缓存] - 内存 - 外存

  • 输出设备

    • 将计算机处理的数据、计算结果等内部信息转换成人们习惯接收的信息形式(字符、图形、声音)，然后将其输出

    • 常用输出设备

      • 显示器

      尺寸：显示区域的对角线距离，单位是尺寸

      接口：

      ​ VGA -> 传统显示接口，15针，传输模拟信号

      ​ DVI -> 数字化的显示接口，传输数字信号

      ​ HDMI -> 数字化传输接口，可同时传输音频和视频信号

      • 打印机

      针式打印机：9、16、24针；适用于银行、收银、票据等，唯一支持多联打印；速度慢、噪音大、质量差、不适合打印图形

      喷墨打印机：将彩色液体油墨经喷嘴变成细小微粒喷到印纸上；适用于家庭或办公使用；打印质量较高， 速度比点阵打印机快、噪音小；但耗材费用高，对纸张要求高，喷墨扣不容易保养

      激光打印机：常用的打印机

      • 音箱、绘图仪和D/A转换器等

指令

指令是指示计算机执行某种操作的命令，由一串二进制码组成

指令包括操作码和操作数(地址码)两部分 操作码规定了操作的类型，进行什么样的操作；地址码规定了要操作的数据(操作对象)存放在什么地址中，以及操作结果存放到哪个地址中去。

程序：为了完成某个任务而设计的一组有序的指令的集合

指令系统

  • 所有指令的集合成为计算机指令系统
  • 复杂指令系统(CISC)
  • 精简指令系统(RISC) 格式统一、种类少、寻址方式少、处理速度快

计算机工作过程

  • 取指令-分析指令-执行指令，指令计数器加1
  • 几乎所有的冯·诺依曼型计算机的CPU，其工作都可以分为5个阶段：取指令、指令译码、执行指令、访存取数、结果写回

计算机软件系统

相关概念

  • 软件：计算机运行所需的程序、数据和有关的文档的综合
  • 程序：解决某一种具体问题的指令序列
  • 数据：数据是程序的处理对象

系统软件

概念：系统软件是指管理、监控和维护计算机资源(包括硬件和软件)、开发应用软件的软件

  • 操作系统：操作系统(OS)是一组对计算机资源进行控制与管理的系统化程序集合
  • 操作系统是用户和计算机硬件系统之间的接口
  • 操作系统为用户和应用软件访问和控制计算机硬件提供了桥梁
  • 作用：操作系统是直接运行在裸机上的最基本的系统软件，任何软件必须在操作系统的支撑；管理计算机、使用计算机
  • 语言处理程序：汇编程序、解释程序、编译程序
  • 程序设计语言[机器语言、汇编语言、高级语言]
    • 程序=数据结构+算法 算法是由有限个步骤组成的用来解决问题的具体过程
    • 数据结构 从问题中抽象出来的数据之间的关系 典型数据结构
  • 数据库管理系统：FoxPro、Access、Oracle、SQL Server等
  • 系统支撑和服务软件 主要指工具软件，如系统诊断程序、调试程序、查杀病毒程序等

指令是指挥机器工作的指示和命令，是能够被机器识别并执行的二进制编码; 指令由操作码和地址码组成。
指令系统是指令的集合，CPU中用来计算和控制计算机系统的一套指令的集合，而每一种新型的CPU在设计时就规定了一系列与其他硬件电路相配合的指令系统。程序是按一定的顺序安排的、用于解决某个具体问题的指令序列。

@@ 计算机的通用性使其可以求解不同的算术和逻辑问题，这主要是取决于计算机的( C )

A.高速运算 B.指令系统 C.可编程性 D.存储功能

考点

  • 裸机：未配置任何软件的计算机叫做裸机，是计算机完成工作的物质基础
  • CPU和内存构成主机
  • 磁盘驱动器和磁带机、触摸屏既是输入设备也是输出设备

@@ 要执行的程序和处理的数据都会从内存直接加载到CPU

@@ 程序退出后，操作系统将回收此程序所占用内存空间

@@ 未运行的程序存放在外存中

@@ 完整的计算机系统应该包括硬件系统+软件系统
