Unity3D+5.0

2022/8/25 建模

Scene 场景编辑 窗口

Q拖动小手 W移动物体 E旋转 R改变物体大小

–右侧的Center是选中所有物体的加权轴心 Pivot是选中物体的单个轴心

按住右键可以移动画面[同时可以WSAD前后左右位移 QE上下位移]

按住ctrl 可以使物体径向移动

Scene下方的 Shaded可以改变所观察物体的方式(显示轮廓 不同通道)

右方的轮廓沙漏轴可以切换视角

Game 游戏运行 窗口

开始 暂停 下一项

下方的Free Aspect可以调整比例

Maximize on Play 游戏下一次运行时将全屏播放

Stats 运行时的参数

Gizmos 光源+粒子效果的标记

Hierarchy 场景物体 列表窗口

可以搜索物体

Project 项目资源 列表窗口

Favorites里可以寻找物体建模

Inspector 属性编辑 列表窗口

Layers为图层类隐藏可视化

其他常用调节窗口

File 文件菜单

  • Build Settings… 设置菜单

    • Run in Background 在运行同时打开其他不干扰
    • Rendning path* 渲染方式

  • Build & Run 项目运行

Edit 编辑菜单

  • preference 对界面的基础设置菜单
  • Selection 当选择物体后可以保存选择的物体 可以随时调出
  • Project Settings
    • Input 游戏的输入管理

Assets 资源菜单 对project资源项目进行的管理

GameObject 物体菜单 对场景Hierarchy的管理

Component 组件菜单 对选中的物体添加一些预设的组件

Window 窗口菜单

  • Animation 对游戏的性能进行测试 也有动画状态机
  • Lighting 光照窗口
  • Navigation 场景角色自动寻路

Help 帮助菜单

Scene 与 场景漫游
