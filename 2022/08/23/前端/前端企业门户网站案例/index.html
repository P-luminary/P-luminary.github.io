2022/8/23 前端

经验技巧总结

  • 在进行页面布局时，位置和对齐，是最重要的基础工作
  • 什么时候出现div：–有改变即添加div[1. 已经改变的 2. 将来可能会改变的]
  • 整体布局时，各个对象的位置、对齐方式、间距等设置要注意细节，要通过计算的方式获取相对位置的对应关系
  • ul 和 li 的使用频率较高，非常适合于规律性显示的页面效果，例如案例中，多次使用到ul 和 li的场景
  • 表格布局一般用 th td， 数据布局一般用ul li
  • 实现水平居中、垂直居中的方法
    • 水平居中：margin：25px auto；
    • 垂直居中：line-height：60px；height：60px
  • 找准层级关系，活学活用div

href=”#” 可以防止网页一直刷新 ###是锚点的效果
用js的时候一定要有name 和 id
var flag = true;
//全局变量只在页面刷新的时候加载一次
onmouseleave()控制鼠标的离开事件
margin: 25px auto [可以自动根据网页分辨率调整 只有两个的话就 上下25px 左右自动]

  • 什么时候出现div：——有改变即添加div
    1.目前已经改变的 和别的格式不一样
    2.将来有可能会产生不同的地方
  • 整体布局时，各个对象的位置、对齐方式、间距等设置要注意细节，要通过计算的方式获取相互位置的对应关系
  • ul和li的使用频率较高，非常适用于规律性显示的页面效果
  • 实现水平居中、垂直居中的方法
    √ 水平居中：margin: 25px auto;
    √ 垂直居中：line-height：60px; height：60px;
  • div活灵活用 在css中不要直接写 要一层一层方便后期维护

        flag=false;
    }else{

    }
}
