网络基础、Internet基础及网页基础

2022/12/14 计算机
  • 计算机网络的概念

==计算机网络是指将地理上分散、具有独立功能的多台计算机，利用通信设备和传输介质相互连接，并配以相应的网络协议和网络软件，实现数据通信和资源共享的计算机系统。==

  • 计算机网络的组成
    • 通信子网：用于提供网络通信功能，完成网络主机之间的数据传输、交换、通信控制和信号变换等工作，主要包括通信线路、网络连接设备、网络协议和通信软件等。
    • 资源子网：由计算机系统(简称主机)、终端控制器和软件数据资源构成，负责网络中的数据处理，向网络用户提供各种网络资源和网络服务。
      • 主机：负责本地或全网的数据处理,运行各种应用程序或大型数据库系统，向网络用户提供各种软硬件资源和网络服务
      • 终端控制器：用于把一组终端连入通信子网，并负责终端信息的接收和发送
      • 终端：终端(Terminal)也称终端设备，是计算机网络中出网络最外围的设备，主要用于用户信息的输入以及处理结果的输出等
    • 计算机系统：主要担任数据处理工作，任务是进行信息的采集、存储和加工处理；大型计算机、微机等
    • 网络节点：网络节点主要负责网络中信息的发送、接收和转发，网络节点是计算机与网络的接口；局域网中使用的网络配置器(网卡)就是网络节点
    • 通信链路：通信线路 → 双绞线、同轴电缆、光纤等传输介质；通信设备 → 中继器(中继器的作用是将数字信号放大)、调制解调器(Modem)[能进行数字信号和模拟信号之间的相互转换，以便数字信号通过只能传输模拟信号的线路来传输]

计算机网络的分类

  • 根据网络的覆盖范围划分

    • **局域网(LAN)**：一个单位、企业的计算机之间为了互相通信而组建

    局域网特点：覆盖的地理范围较小、传输速率搞、通信延迟时间短、可靠性较高

    特点：链接范围窄、用户数少、配置容易、连接速率高

    技术：以太网、WiFi

    • **城域网(MAN)**：是在一个城市范围内所建立的计算机通信网
    • 广域网(WAN)：用于链接不同城市之间的LAN或MAN；是连接不同地区局域网或城域网计算机通信的远程王。广域网不等同于互联网，Internet是最大的广域网
    • 个人局域网(PAN)：短距离无线通信技术；实现技术：Bluetooth、IrDA、热点

  • 按网络的拓扑结构划分

    • 总线型拓扑

      • 优点: 结构简单，布线容易、站点扩展灵活方便，可靠性高。
      • 缺点: 故障检测和隔离较困难，总线负载能力较低。

    • 星型拓扑[局域网最多]

      • 优点: 传输速度快、误差小、扩容比较方便、易于管理和维护，故障的检测和隔离也很方便。
      • 缺点: 中央节点是整个网络的瓶颈，必须具有很高的可靠性，中央节点一旦发生故障，整个网络机会瘫痪，耗费电缆多

    • 环形拓扑

      • 优点: 容易安装和监控，传输最大延迟时间是固定的，传输控制机制简单，实时性强。
      • 缺点: 网络中任何一台计算机故障都会影响整个网络的正常工作,故障检测比较困难，节点增删不方便

    • 树形拓扑

      • 优点: 在扩容和容错方面有很大优势,容易将错误隔离在小范围内

      • 缺点: 依赖根结点,根结点出故障整个网络会瘫痪

        对星型结构的扩展，按层次进行连接，相邻及同层结点之间一般不进行数据交换或数据交换量小。

    • 网状型拓扑

      • 通常用于广域网中，优点是节点间路径多，可靠性高，扩充入网灵活方便。缺点是结构和协议复杂，建网成本高

  • 按传输介质划分

    • 有线网
      • 双绞线[局域网4组8根] 100m以为改光纤/或者加入中继器加强范围
      • 同轴电缆 容易安装、造价较低、网络抗干扰能力强、维护或扩展困难、断点较多影响系统的可靠性
      • 光纤 光导纤维，传输距离元，传输率搞，抗干扰能力强，安全好用，成本高；光信号-电信号(模拟信号、数字信号)
    • 无线网：主要以无线电波或红外线为传输介质 另外还有卫星数据通信
      联网方式灵活方便，但联网费用较高，安全性和可靠性还需完善
      • 短距离通信：WiFi、WLAN[无限局域网]、Bluetooth、Zigbee
      • 移动网络通信：4G、5G
      • 卫星通信：美国GPS、中国北斗卫星、欧洲Galileo伽利略、俄国GLONASS

  • 性能指标

    • 速率：数据传输的有效性指标

    单位时间内从网络中的一个点到另一个点所能通过的网络数据率，带宽的单位是b/s

    每秒钟传输的二进制位，单位是比特每秒，记作b/s Mb/s=10^6b/s[物理单位] kb/s=10^3b/s

    • 带宽：在模拟信道中，用带宽表示信道传输信息的能力，即通信信道允许通过信号频率的范围，体现为最高频率和最低频率之差，单位是赫兹(Hz)
      在计算机网络中，带宽表示在单位时间内网络中某信道所能通过的最高数据率，单位bit/s
    • 误码率：数据传输的可靠性指标
      误码率是指二进制码元在传输系统中传错的概率，用”传错的码元数/所传码元的总数”来表示
    • 时延：数据从网络的一端传送到另一端所需要的时间 延迟
    • 信道：传输信息的通道

  • 计算机网络功能

    • 信息交流
    • 资源共享：资源共享是计算机网络的主要功能
      计算机资源主要是指计算机的硬件、软件和数据资源
    • 分布式处理：负载均衡
    • 提高系统的安全与可靠性：计算机通过网络中的冗余部件，尤其是借助虚拟化技术可大大提高网络可靠性

  • 网络接入设备

    • MODEM
      • 调制解调器，实现模拟信号和数字信号的相互转换
      • 调制 数字信号→模拟信号 D/A转换
      • 解调 模拟信号→数字信号 A/D转换
    • 网卡：网络适配器，是OSI/RM的数据链路层设备
      • MAC地址，每个网卡都有唯一的标识
      • MAC地址用于网络中的唯一标示一个网卡，一台设备若有一个或多个网卡，则每个网卡都需要并会有一个唯一的MAC地址
      • MAC地址的长度为48位(6个字节)二进制，通常标识为12个16进制数；前六位16进制数00-16-EA代表网络硬件制造商的编号。后六位16进制数AE-3C-40代表该制造商所制
      • 造的某个网络产品(如网卡)的系列号。MAC地址在世界是唯一的

  • 网络体系结构—OSI

    概念：计算机网络体系结构就是计算机网络的层次结构模型和层间协议的集合

    网络协议是网络中的计算机为进行数据交换而制定的规则、标准或约定

    • OSI参考模型

      **OSI：开发系统互连参考模型 **[巫术网传会表演]

      • 应用层：DNS、HTTP、SMTP、DHCP、FTP等，每个应用层协议都是为了解决某一类应用问题
      • 表示层
      • 会话层
      • 传输层：传输数据单位是传输数据单元TPC/IP协议
      • 网络层
      • 数据链路层：通信设备有交换机、网络适配器，传输数据单元是帧
      • 物理层：中继器、集线器、传输比特流

    • TCP/IP协议模型对应各层协议图[赢输往往=ysww]

      TCP/IP协议模型 各层对应协议
      应用层 Telnet、FTP、HTTP、SMTP、DNS、RTP、SNMP、NFS
      传输层 传输控制协议TCP、用户数据报协议UDP
      网络层(网际层) 网络协议IP、ICMP、IGMP、ARP
      网络接口层 Ethernet、Token Ring、Fame Relay、ATM

      传输层：传输层为应用进程之间提供端到端的逻辑通信

      TCP协议：传输控制协议，提供可靠的、面向连接的服务

      服务器端各应用进程端口号：ftp21，telent 23，DNS 53，http 80，https 443

    • 数据交换方式

      • 电路交换：线路时延小，但是线路利用率低，比如电话
      • 报文交换：线路可靠性高、线路利用率高、不适合于会话型和实时性要求较高的业务。E-mail
      • 分组交换：线路利用率高。计算机网络

  • 接入Internet常用方式

    • 拨号上网方式：动态分配一个合法的IP地址
      特点: 投资上,适合一般家庭及个人用户使用;速度慢，受电话线及接入设备限制,速率一般是56Kb/s

    • ISDN专线接入：一线通/窄带综合业务数字网
      是在现有电话网上开发的一种集语音、数据和图像通信于一体的综合业务模式，,可以边打电话边上网
      缺点: 1.速度相对于ADSL和LAN等接入方式来说,速度不够快。2.长时间在线费用会很高。3.设备费用并不便宜。
      特点: 方便,速度快,最高上网速度可达128Kb/s

    • ADSL宽带入网：[用户端有一个动态分配的IP地址] ADSL(Asymmetrical Digital Subscriber Line,非对称数字用户线路）是一种能够通过普通电话线提供宽带数据业务的技术。计算机-ADSL Modem-分离器-PSTN-Internet
      特点：利用普通铜制电话线作为传输介质，配上Modem,不需要改造信号传输线路。
      ==上行速率640Kb/s ~ 1Mb/s，下行速率1 ~ 8Mb/s，有效传输距离3~5km==

    • DDN专线接入：利用数字传输通道(光纤、数字微波和卫星）和数字交叉复用节点所组成的数字数据传输网
      特点: 采用数字电路，传输质量高，时延小,通信速率可根据需要选择;可靠性高

    • 局域网接入：把用户的计算机连接到一个与因特网直接相连的局域网LAN上，并且由一个IP地址。特点 传输速率高，需要网卡，设置TCP/IP

  • 通信子网：通信子网提供计算机网络的通信功能，由网络节点和通信链路组成

    ​ 通信设备

    • 物理层设备
      • 中继器(repeater)：中继器适用于完全相同的两类网络的互连，主要功能是通过对数据信号的重新发送或者转发来放大信号，是局域网下用来扩大网络传输距离的设备
      • 集线器(hub)：集线器工作在OSI/RM的物理层，主要功能是对接收到的信号进行再生整形放大，以扩大网络的传输距离；由于集线器会把收到的任何数字信号，经过再生或放大，再从集线器的所有端口提交，这会造成信号之间碰撞的机会很大,而且信号也可能被窃听,并且这代表所有连到集线器的设备,都是属于同一个碰撞域名以及广播域名,因此大部份集线器已被交换机取代。
    • 数据链路层设备
      • 网桥(bridge)：网桥，又叫桥接器，工作在OSI/RM的数据链路层，连接两个局域网，像聪明的中继器,可识别MAC,并隔离冲突域。已被具有更多端口的交换机取代。
      • 交换机(switch)：工作在OSI/RM的数据链路层，多点连接，根据MAC地址寻址，隔离冲突域
    • 网络层设备
      • 路由器(router)：路由器根据IP地址寻址，可以处理TCP/IP协议。路由器的一个作用是连通不同的网络，另一个作用是选择信息传送的线路，连接异种网络的设备
    • 交换机和路由器的区别

    交换机工作在数据链路层，路由器工作在网络层
    交换机根据MAC地址寻址，路由器根据IP地址寻址
    交换机扩大局域网接入点，路由器是连接不同网络
    交换机可以把很多主机连接起来,对外的IP不同。路由器可以把—个IP分给多个主机使用，对外IP相同。

    家用路由器是**路由器+交换机**的集合

  • Internet组成

    • 资源子网：资源子网提供访问网络和处理数据的能力，由主机、终端控制器和终端组成
    • 资源共享方式
      • C/S模式：Client/Server，客户端/服务器模式
      • B/S模式：Browser/Server，浏览器/服务器模式
      • P2P模式：Peer to Peer，对等网络，P既是服务器也是客户端，如迅雷等下载软件

  • IP地址

    • 概念：在Internet上为每台计算机指定的唯一的32位地址称为IP地址，也称为网际地址，全球唯一

    • 版本

      • ==IPV4 ($2^{32}$)32位二进制数组成，分成4段，每8位一段==
        IP地址以十进制形式表示，点分十进制，三个点四组数
        每组数的取值范围是0 ~ 255 如 202.119.2.199
        判断合法IP地址的标准：1. 三个点分成四组数 2.每组数的取值范围是0 - 255
      • IPV6 ($2^{128}$)128位二进制数组成，一劳永逸地解决了IP地址短缺的问题
        IPV6地址以冒号分十六进制表示，7个冒号分8组十六进制数,每组为四个十六进制数的形式,如2001:0DB8:85A3:0000:1319:8A2E:0370:7344
        优势：规模更大、更安全可信、更及时、更方便、更可管理、更有效，速度更快,
        100Mb/s以上

  • IP地址获取 手动设置静态IP / DHCP服务器分配自动获得

    • 默认子网掩码

      子网掩码是判断任意两台计算机的IP是否属于同一子网的依据。最简单的方法是将两台计算机各自的IP地址与子网掩码进行AND(与)运算，如果得出的结果是相同的，则属于同一子网，可以直接通信。子网掩码不能单独存在，它必须结合IP地址一起使用

      • A类IP地址 255.0.0.0
      • B类IP地址 255.255.0.0
      • C类IP地址 255.255.255.0

      可变长子网掩码VLSM

      • 作用：节省IP地址资源、便于管理、实现网络的层次性

        满足不同网络对IP地址的需求

      考法：
      1.子网掩码有什么用？ 划分子网
      2.子网掩码1和0各代表什么？ 1代表网络位 0代表主机位
      3.A类 B类 C类IP地址的默认子网掩码是什么？
      4.已知IP地址及子网掩码
      求网络地址和广播地址；
      网络地址：IP地址和子网掩码求与运算
      192.168.0.1 与 192.168.0.**2 **是同网段主机的IP{主机位全0的是网络地址}{网络地址不能分配给主机使用0/255 只能1-254}
      广播地址：网络地址的网络位不变，主机位全变成1 192.168.0.255[ 1111 1111]
      求子网数及每个子网可用主机数；
      子网数：每个网络位占用了几个主机位
      判断一下IP地址是否为同一子网，两个IP地址分别与子网掩码求与运算

  • 域名

    为了方便用户，Internet在IP地址的基础上提供了一种面向用户的字符型主机命名机制，这就是域名系统
    IP地址是Internet主机的作为路由寻址用的数字体标识，人不容易记忆。因而TCP/IP协议专门设计了域名这一种字符型主机命名机制，即给每一台主机一个有规律的名字(由字符串组成)

    • 域名结构：IP地址的助记符，也用英又的句号（小数点)分隔，最后一个小数点的石侧是一级域名（或称顶级域名），从右往左依次（二级域名、三级域名等)，域名全球唯一 比如www.qq.com, www是主机名; 确认键网课地址zsb.querenjian.com, zsb是主机名

    • 域名系统的规定

      • 组织域名

        ==.com商业组织，**.edu教育机构，.gov政府部门，.mil军事部门，.org非营利性组织**==等。
        组织域名是按组织管理的层次结构划分所产生的组织型域名，由三个字母组成

      • 国家或地区域名

        .cn是中国顶级域名，.jp是日本顶级域名等
        地理模式按国别地理区域划分，这类域名是世界各国或地区的名称，并且规定由两个字母组成

    • 域名系统(DNS) （Domain Name System，缩写:DNS）是互联网的一项服务。它作为将域名和IP地址相互映射的一个分布式数据库，能够使人更方便地访问互联网。
      DNS服务器端口号53，主要用来域名解析使用
      当前，对于每一级域名长度的限制是63个字符，域名总长度则不能超过255个字符

    • 域名管理机构

      中国互联网络信息中心(CNNIC) 管理除edu以外的其它域名
      中国教育和科研计算机网络中心 管理edu域名

    • IP地址和域名的关系

      一个域名只能对应一个IP地址
      一个IP地址可以对应多个域名 querenjian.com querenjian.cn

  • Internet主要服务

    • 电子邮件服务E-mail

      • 格式：用户名@电子邮件服务器名 → 390415030@qq.com ← 登录名@主机名.域名
      • 协议：SMTP、POP3、MIME协议
      • 典型C/S模式，常用客户端有：Outlook Express、Foxmail等
      • 收件人：多个收件人用分号隔开

    • 文件传输协议FTP

      • 登录ftp方式：浏览器或资源管理器登录 1.打开浏览器，在地址栏中输入ftp://服务器IP地址。然后回车。2.这时候会出现一个验证界面，需要输入分配的用户名和密码，输入后点击登录(如果需要可以勾选匿名登录)。3.进入了FTP服务器中，就可以进行文件的上传和下载了。ftp客户端登录 filezilla、Fireftp等

    • 远程登录Telent

      • 用户通过一台计算机的Telnet客户程序，采用命令行的方式登录访问另一台计算机上的Telnet服务器，从而运行另一台计算机上的程序并访问其中的服务
      • talent远程登录协议 默认端口号23
      • Telnet、SSH、TEAMViewer、向日葵远程控制等

    • 万维网WWW

      • World Wide Web

      • URL(Uniform Resource Locator)统一资源定位器，因特网的万维网服务程序上用于指定信息的表示方法

      • 语法 → 协议名://主机域名:端口号/路径/文件名
        https://zsb.querenjian.com:80/index.html

      • 协议名：输入时可省略，默认是http(超文本传输协议); 还可用https(http加密)、ftp(文件传输协议)

      • 类型

        相对URL：就是相对于自己的目标文件位置 在相对路径中经常使用 “../“ 来表示上一级目录
        < body background = “../bg.jpg” >

        绝对URL：是指文件在硬盘上真正存在的路径

    • B/S模式(浏览器)

      • WWW采用浏览器/服务器模式

    • 常见的下载方式

      • 浏览器下载、http下载(不支持断点续传)、ftp下载(类似于网盘，适合大文件传输)、p2p下载(点对点下载)、p2sp下载(迅雷)、流媒体下载(支持多线程与断点续传)

    • 信息浏览与检索

      • Web服务器软件 Microsoft IIS、Apache和Tomcat
        LAMP组合(Linux、Apache、MySQL、PHP)

    • 网页设计语言

      • HTML扩展名为.htm和.html
      • XML 可扩展标记语言 在Internet上传递或处理数据
      • CSS 层叠样式表 对网页数据进行编排、格式化、显示、设置特效等
      • DHTML 动态HTML
      • HTML5 具有自适应网页设计，支持多设备跨平台使用，为桌面和移动平台带来无缝衔接的丰富内容等诸多优点
        HTML5 设计目的是为了在移动设备上支持多媒体，如Video、Audio和Canvas等标记。
      • 脚本语言
        脚本(Script)语言是嵌入到HTML代码中的程序，根据运行的位置不同把它分为客户端脚本和服务器脚本。
        脚本语言JavaScript更多应用于客户端（浏览器)，VBScript更多用于服务器端

    • 头部(Head)

      • < meta > 在百度排名会增加；在HTML语言中，包含关键字，网页描述信息等内容的标记是meta
        浏览器访问安全性要求较高的网页时，HTTP会调用SSL/TLS对网页进行加密,地址栏里http变成https

        规定HTML文档的字符编码 < meta charset = “utf-8” >

        为搜索引擎定义关键词 < meta name = “keywords” content = “专升本” >

        为网页定义描述内容 < meta name = “description” content = “ 统招专升本考试备考平台 “ >

        定义网页作者 < meta name = “author” content = “升本吴彦祖” >

        每30秒钟刷新当前页面 < meta http-equiv=”refresh” content = “30” >

        作用：META元素通常用于指定页面的描述，关键词，文件的最后修改时间，作者和其他元数据。在HTML中< meta >标签没有结束标签

    • < img >标记插入图片

      • src属性 指明图片文件所在的位置 < img src = URL >

      • alt属性 图片的文字说明，鼠标指向图片时弹出说明性文字 < img src = URL alt = 说明文字 >

      • align属性 用于设置图片相对于文本的位置关系 < img src = RUL align = 对齐方式 >

        对齐方式可以是：top(顶端对齐)、middle(相对垂直居中)、bottom(相对底边对齐)、left(左对齐)、right(右对齐)、texttop(文字上方)

      • < a >超链接

        文本格式：< a href = URL > 超链接文本 < /a >

        图片格式：< a href = URL1> < img src = URL2 > < /a >

      • < table > 表格

        定义表格区域 < table > … < /table>

        定义表格标题 < caption > … < /caption >

        定义表格行 < tr > … < /tr>

        定义表格头 < th > … < /th>

        定义表格单元格 < td > … < /td >
